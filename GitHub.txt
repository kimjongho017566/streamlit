로컬 환경에서 터미널(명령 프롬프트)이나 Git Bash를 실행하고, 프로젝트 폴더로 이동합니다.
다음 명령어를 사용하여 로컬 저장소를 GitHub에 연결하고 코드를 푸시합니다.
bash
코드 복사
git init  # 프로젝트 폴더에서 git 초기화
git add .  # 모든 파일 추가
git commit -m "Initial commit"  # 커밋 메시지 작성
git branch -M main  # 브랜치를 main으로 변경
git remote add origin https://github.com/kimjongho017566/streamlit.git  # GitHub 저장소 URL 추가
git push -u origin main  # 코드를 GitHub에 푸시
여기서 username은 GitHub 사용자명, repository_name은 저장소 이름으로 바꿔야 합니다.

4. Streamlit Cloud로 배포
Streamlit Cloud를 사용하면 GitHub에 있는 앱을 쉽게 배포할 수 있습니다.

Streamlit Cloud에 접속하여 계정으로 로그인합니다.
로그인 후 "New app" 버튼을 클릭합니다.
GitHub 저장소를 선택하고, 배포할 브랜치와 파일(app.py)을 선택합니다.
"Deploy" 버튼을 클릭하면 자동으로 배포가 진행됩니다.
앱이 배포되면 URL이 제공되며, 이를 통해 누구나 앱에 접근할 수 있습니다.
